## Installation

> Need have **npm** installed [Install NPM](https://nodejs.org/en/)

## Usage

Install dependencies (Tailwind)

```
  npm install
```

Run the commando bello to generate a file named "alunos_transferidos.xlsx" with the transfer studants list

```
  npm run dev
```
